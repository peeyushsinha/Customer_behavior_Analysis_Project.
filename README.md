## Customer Shopping Behavior Analysis

This project demonstrates an end-to-end Data Analytics workflow using a Kaggle dataset focused on customer shopping behavior. The objective is to analyze purchasing patterns, customer segments, and buying trends to generate meaningful business insights and support data-driven decision-making.

## Project Overview

✅ Data Preparation, Modeling & Exploratory Data Analysis (Python):
  .Cleaned and transformed raw data for analysis
  .Performed data preprocessing and feature engineering
  .Conducted Exploratory Data Analysis (EDA) to identify trends, patterns, and customer behavior insights

✅ Data Analysis (SQL):
   .Simulated business-oriented analytical queries
   .Extracted insights related to customer segmentation, loyalty, and purchasing behavior
   .Analyzed sales trends and key purchase drivers using SQL queries
   
✅ Visualization & Business Insights (Power BI):
   .Developed an interactive Power BI dashboard
   .Visualized customer trends, shopping patterns, and business KPIs
   .Enabled stakeholders to make informed, data-driven decisions through dynamic reports and insights
